#  Interactive Task Dashboard

##  Project Overview

The **Interactive Task Dashboard** is a frontend React application built to demonstrate state management, dynamic rendering, and clean component architecture. 

This project allows users to create, manage, and track tasks in an interactive dashboard interface. It uses React functional components, React hooks, and browser localStorage to provide a smooth and persistent user experience.

---

##  Problem Statement

Build an interactive dashboard using React that demonstrates:

- State management using `useState`
- Side effects using `useEffect`
- Dynamic rendering of components
- Clean component-based architecture
- Dark/Light theme toggle functionality
